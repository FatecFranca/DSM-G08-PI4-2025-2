# Mobile DSM-G08-PI

Aplicativo mobile feito em **React Native com Expo**.  
Consome a API presente na pasta `API/` do mesmo projeto.

## Como rodar:
1. Vá até a pasta Mobile:
   ```bash
   cd Mobile
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Execute com Expo:
   ```bash
   npx expo start
   ```
4. Escaneie o QR Code com o **Expo Go** no celular.
